Start screen share

  1. Click the + (new tab) button in the session tab bar.
  2. In the URL box row, click the screen share icon (looks like a Wi-Fi / broadcast symbol).
  3. Select the window or screen you want to share, then click Share.
Note: Screen sharing must be initiated via the in-session UI for user consent.
